Apa Itu Programmer?

Programmer adalah orang yang menulis, menguji, debug, dan memelihara kode komputer. Mereka menerjemahkan kebutuhan manusia ke dalam bahasa yang dimengerti mesin.

Tipe Programmer

  • Frontend Developer: Fokus pada tampilan dan interaksi pengguna (HTML, CSS, JS).
  • Backend Developer: Menangani logika server, database, dan API (Python, PHP, Java).
  • Full Stack Developer: Menguasai frontend dan backend.
  • Mobile Developer: Membuat aplikasi iOS/Android (Swift, Kotlin).
  • Data Scientist: Analisis data dan machine learning (Python, R).
  • DevOps Engineer: Otomasi infrastruktur dan deployment.

Apa Itu Coding?

Coding adalah proses menulis instruksi dalam bahasa pemrograman untuk memberitahu komputer apa yang harus dilakukan. Ini inti dari programming.

Konsep Dasar Coding

  • Variabel: Tempat menyimpan data (x = 5).
  • Fungsi: Blok kode yang bisa dipanggil berkali-kali.
  • Kondisional: Percabangan (if-else, switch).
  • Loop: Perulangan (for, while).
  • Struktur Data: Array, object, list, dictionary.
  • Algoritma: Langkah-langkah logis menyelesaikan masalah.

SQL (Structured Query Language)

Fungsi: Mengelola dan memanipulasi database relasional.

Penggunaan: Query data, CRUD operations, database administration.

-- SQL Modern (CTE, Window Functions)
WITH user_purchases AS (
    SELECT 
        u.id,
        u.name,
        COUNT(o.id) as total_orders,
        SUM(o.amount) as total_spent,
        RANK() OVER (ORDER BY SUM(o.amount) DESC) as rank
    FROM users u
    LEFT JOIN orders o ON u.id = o.user_id
    WHERE u.created_at >= '2024-01-01'
    GROUP BY u.id, u.name
    HAVING COUNT(o.id) > 0
)
SELECT 
    id,
    name,
    total_orders,
    total_spent,
    CASE 
        WHEN total_spent > 1000 THEN 'VIP'
        WHEN total_spent > 500 THEN 'Premium'
        ELSE 'Standard'
    END as customer_tier
FROM user_purchases
WHERE rank <= 10
ORDER BY total_spent DESC;

-- Index creation for optimization
CREATE INDEX idx_user_email ON users(email);
CREATE INDEX idx_order_date ON orders(order_date DESC);

-- Transaction with rollback
BEGIN TRANSACTION;
UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;
COMMIT;

Google Apps Script (GS)

Fungsi: JavaScript-based untuk otomasi Google Workspace.

Penggunaan: Otomasi Sheets, Docs, Gmail, Forms, Calendar.

// Google Apps Script - Complete Automation Example
function onFormSubmit(e) {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const lastRow = sheet.getLastRow();
    const data = sheet.getRange(lastRow, 1, 1, sheet.getLastColumn()).getValues()[0];
    
    // Send email notification
    const recipient = data[1]; // Email from form
    const name = data[0];
    const timestamp = data[2];
    
    GmailApp.sendEmail(
        recipient,
        'Form Submission Confirmation',
        `Hello ${name},\n\nYour form submitted at ${timestamp} has been processed.`,
        { name: 'Auto-System' }
    );
    
    // Create Google Doc report
    const doc = DocumentApp.create(`Report_${name}_${timestamp}`);
    const body = doc.getBody();
    body.appendParagraph(`Report for ${name}`)
        .setHeading(DocumentApp.ParagraphHeading.HEADING1);
    
    body.appendParagraph(`Email: ${recipient}`)
        .setBold(true);
    
    // Add to Drive folder
    const folder = DriveApp.getFolderById('FOLDER_ID_HERE');
    DriveApp.getFileById(doc.getId()).moveTo(folder);
    
    // Log activity
    Logger.log(`Processed submission for ${name} at ${new Date()}`);
}

// Auto-trigger setup
function createTrigger() {
    ScriptApp.newTrigger('onFormSubmit')
        .forSpreadsheet(SpreadsheetApp.getActive())
        .onFormSubmit()
        .create();
}

// Contoh API endpoint dengan Python Flask
from flask import Flask, request, jsonify
import mysql.connector

app = Flask(__name__)

@app.route('/api/users', methods=['POST'])
def create_user():
    data = request.json
    # Koneksi database
    db = mysql.connector.connect(
        host="localhost",
        user="admin",
        password="secret",
        database="app_db"
    )
    cursor = db.cursor()
    
    # Insert data
    sql = "INSERT INTO users (name, email) VALUES (%s, %s)"
    values = (data['name'], data['email'])
    cursor.execute(sql, values)
    db.commit()
    
    return jsonify({"message": "User created", "id": cursor.lastrowid})

if __name__ == '__main__':
    app.run(debug=True)
